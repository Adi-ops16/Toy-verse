## Purpose
This is a modern and responsive **Toy Store** web application built with **React** and **Tailwind CSS**. Users can explore toys, view featured products, and manage authentication securely with Firebase. The website is designed for a smooth, interactive shopping experience.

## Key Features
- **Responsive Design:** Works seamlessly on desktop, tablet, and mobile.
- **Firebase Authentication:** Register, login, and reset passwords securely.
- **Featured Toys Section:** Highlights popular toys dynamically.
- **Interactive UI:** Smooth animations with **AOS** and **Swiper**.
- **Charts & Analytics:** Visualize toy-related data with **Recharts**.
- **Notifications:** User-friendly alerts using **SweetAlert2**.
- **Reusable Components:** Modular design for easy scalability.

## npm Packages Used
- **react**: Frontend UI library  
- **react-dom**: DOM rendering for React  
- **react-router**: Client-side routing  
- **firebase**: Backend services and authentication  
- **tailwindcss**: Utility-first CSS framework  
- **@tailwindcss/vite**: Tailwind integration with Vite  
- **aos**: Scroll animations  
- **lucide-react**: Icon library  
- **react-spinners**: Loading spinners  
- **recharts**: Data visualization  
- **sweetalert2**: Alerts and notifications  
- **swiper**: Modern sliders/carousels
